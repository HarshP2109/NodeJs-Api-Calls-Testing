# Worko API

## Getting Started

### Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/HarshP2109/NodeJs-Api-Calls-Testing.git
   cd NodeJs-Api-Calls-Testing
    ```

2. Install dependencies:
    ```sh
    npm install
    ```

3. Setup Server:
    ```sh
    npm start
    ```

4. Run Test:
    ```sh
    npm test
    ```
